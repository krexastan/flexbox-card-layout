Objective
Design a responsive card layout using CSS Flexbox

Your webpage must include:
•	Proper HTML 5 structure
•	One main container element
•	Six child elements inside the container
•	display: flex
•	flex-wrap
•	justify-content
•	align-items
•	gap
•	flex property
•	A hover effect

Task Guidelines
1.	Add a clear main heading at the top of the page.
2.	Create a container below the heading.
3.	Inside the container, create six cards.
4.	Each card must contain:
- One image
- One title
5.	Arrange the cards in rows using Flexbox.
6.	Allow the cards to wrap onto the next line on smaller screens.
7.	Add equal spacing between the cards
8.	Add a hover effect that slightly enlarges each card.
9.	Ensure the entire layout is centered and responsive.

Design Requirements
•	Use consistent spacing throughout the layout.
•	Add rounded corners to each card.
•	Add a light shadow effect to the cards.
•	Keep the design clean and simple.
